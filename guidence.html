Welcome to Code Canva – User Guide

🎯 What is Code Canva?

Code Canva is an interactive platform for learning HTML, CSS, and JavaScript by typing code directly into a beautiful canvas. You’ll see real-time output as you write code, helping you understand how each line works instantly.

🛠️ How to Use It

  • Step 1: Click on the respective tab – HTML, CSS, or JavaScript.
  • Step 2: Type or paste your code into the editor area.
  • Step 3: The output is automatically updated in the preview window below.
  • Step 4: You can copy your code using the Copy button provided.
  • Step 5: Use it to experiment, learn, or quickly prototype your ideas!

💡 Features

  • ✨ Live preview of HTML, CSS & JS.
  • 🖋 Clean UI with dark theme for reduced eye strain.
  • 📋 Copy code with one click.
  • 📱 Responsive design – works great on desktop & mobile.

📎 Tips & Tricks

  • Use HTML to structure your page – head, body, div, etc.
  • Use CSS to style your page – color, font, margin, etc .
  • Apply CSS styles like colors, margins, and fonts in the CSS tab.
  • Write interactivity in JavaScript – try `alert("Hello!")` or `console.log()`!
  • Refresh to reset everything.

🔗 Useful Links

  • MDN HTML Docs
  • MDN CSS Docs
  • MDN JS Docs

🙌 Happy Coding!

🎮 Game Mode – Guide & Instructions

Welcome to Code Canva Game Mode – an interactive way to master HTML, CSS, and JavaScript with real-time challenges!

🕹️ How to Play

  • Step 1: Read the code challenge or instruction displayed above the input box.
  • Step 2: Type the correct code in the editor field below it.
  • Step 3: If your input matches the correct answer exactly, the game automatically advances to the next level.
  • Step 4: Repeat and improve your accuracy!

🧠 Features

  • ✅ Real-time validation of typed code
  • ⚡ Instant transition to the next challenge
  • 🌟 Great for memorizing HTML/CSS/JS basics
  • 📱 Responsive and optimized for all devices

💡 Tips

  • Code must match the required output exactly – spacing, tags, and punctuation matter.
  • Use this mode to reinforce core concepts through repetition.
  • Try completing all levels without errors for better retention.

📁 Projects Page – Guide & Instructions

The Projects Page on Code Canva is where you can test, view, and interact with your saved or demo code snippets using HTML, CSS, and JavaScript.

🛠️ How to Use This Page

  • Step 1: Scroll through the listed projects or templates shown on the page.
  • Step 2: Click on any project title or button to open and test it live in a sandbox-like environment.
  • Step 3: Experiment with the structure, style, and scripts inside the interactive editor.
  • Step 4: Some templates are editable, so feel free to modify them and preview changes.

📌 Purpose of Projects Page

  • 🔍 View working examples of HTML/CSS/JS components.
  • 🧪 Use as templates for practicing or building your own versions.
  • 📚 Learn by reading and tweaking well-structured code samples.

💡 Tips

  • Start with simple projects to understand layout and structure.
  • Compare CSS styling and HTML tags to see how they work together.
  • Use browser developer tools for deeper inspection and learning.
